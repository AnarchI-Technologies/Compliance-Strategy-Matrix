# Strategic Roadmap

This document is a planning aid, not legal, tax, banking, securities, or compliance advice.

## Phase 1: Public-Safe IP Foundation

Document AnarchI systems as software IP first: architecture, source control, test evidence, proof ledgers, and clear ownership records.

## Phase 2: Control Design

Define controls before connecting live financial, customer, or operational systems:

- source-of-funds documentation
- approval gates for write-capable actions
- audit tokens for material events
- separation between public examples and private runtime state

## Phase 3: Professional Review

Before public launch or regulated activity, route the operating model through qualified counsel and accounting review.

## Phase 4: Evidence Preservation

Keep dated proof records, hashes, and commit references in the local AnarchI-IP ledger.
